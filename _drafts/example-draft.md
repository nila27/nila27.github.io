이 폴더(`_drafts/`)에 둔 글은 **빌드되지 않아 라이브 사이트에 뜨지 않습니다.**
단, 이 저장소가 Public이므로 GitHub 저장소 페이지에서 마크다운 원문은 보입니다.

발행 준비가 되면 파일을 `_posts/` 폴더로 옮기고 파일명을
`YYYY-MM-DD-제목.md` 형식으로 변경하면 자동 발행됩니다.

로컬에서 드래프트 미리보기:
```
bundle exec jekyll serve --drafts
```
